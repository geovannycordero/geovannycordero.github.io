10:T8c1,<p>How to install ruby on Mac OS Sonoma with the M2 chip?</p>
<p>Recently, I was working on a project where I require to use an old version of ruby, it was ruby <code>2.7.4</code> . I try to
install this version using <code>rvm</code> and <code>rbenv</code> without success.</p>
<p>After some research I finally found a tool that works for my and the ruby version I was trying to install, this tool is
called <code>asdf</code> .</p>
<h3>How to install asdf</h3>
<p>There are different ways to get <code>asdf</code> listed in the <a href="https://asdf-vm.com/guide/getting-started.html">official page</a>,
but I did it using git.</p>
<p>Open your terminal and run the following command:</p>
<pre><code class="language-bash">git clone https://github.com/asdf-vm/asdf.git ~/.asdf --branch v0.14.0
</code></pre>
<p>Then, to install it there is again a lot of different ways, but I decided to add it as a plugin to my previous installed
program oh-my-zsh (that I highly recommend to use in mac).</p>
<p>Just add it to to the <code>plugins</code> definition in the <code>~/.zshrc</code> . In my case I was already using the <code>git</code> plugin so it
looks as follows, separated by an space:</p>
<pre><code class="language-bash">plugins=(git asdf)
</code></pre>
<p>You can check the official instructions <a href="https://github.com/ohmyzsh/ohmyzsh/tree/master/plugins/asdf">here</a>.</p>
<p>Close and open a new terminal or run <code>source ~/.zshrc</code> to update your configuration and you should be able to use <code>asdf</code>
now.</p>
<h3>Ruby with <code>asdf</code></h3>
<p>First, add ruby as a plugin and then install the required version</p>
<pre><code class="language-bash">$ asdf plugin add ruby
$ asdf install ruby 2.7.4
$ asdf global ruby 2.7.4

# Update to the latest Rubygems version
$ gem update --system
</code></pre>
<p>Confirm that you have ruby installed:</p>
<pre><code class="language-bash">$ which ruby
~/.asdf/shims/ruby
$ ruby -v
ruby 2.7.4p191 (2021-07-07 revision a21a3b7d23) [arm64-darwin23]
</code></pre>
<p>Install rails</p>
<pre><code class="language-bash">asdf exec gem install rails
</code></pre>
<p>You can also install NodeJS using <code>asdf</code> but for now I’ll continue using <code>nvm</code>.</p>
